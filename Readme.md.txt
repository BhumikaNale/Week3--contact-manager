Week 3: Contact Management System

Project Description
The Contact Management System is a Python-based console application that allows users to store, manage, and organize contact information efficiently. Contacts are stored persistently using a JSON file.

Features
- Add new contacts
- Search contacts by name
- Update existing contacts
- Delete contacts
- Display all stored contacts
- Data persistence using JSON

Technologies Used
- Python 3
- JSON
- File Handling
- Dictionaries
- Functions
